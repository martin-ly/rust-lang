# Crates 文档标准规范

> **日期**: 2025-11-15
> **版本**: v1.0

---

## 文档标准

### 格式规范

1. **文件命名**: 使用小写和下划线，如 `01_quick_start.md`
2. **标题格式**: 使用 `#` 到 `####` 四级标题
3. **日期标注**: 所有文档须标注最后更新日期
4. **版本标注**: 标注适用的 Rust 版本

### 内容结构

```markdown
# 文档标题

> **最后更新**: YYYY-MM-DD
> **适用版本**: Rust X.Y.Z+

---

## 目录

- [章节1](#章节1)
- [章节2](#章节2)

---

## 章节1

内容...

---

## 相关文档

- [链接](./README.md)
```

### 链接规范

- 内部链接使用相对路径
- 从 `docs/` 引用 `crates/` 使用 `../crates/`
- 确保所有链接有效

---

## 质量检查清单

- [ ] 文档有清晰的标题和描述
- [ ] 包含目录导航
- [ ] 代码示例可编译
- [ ] 所有链接有效
- [ ] 已标注日期和版本
