## ğŸ”¬ Formal Verification

This project includes comprehensive formal verification tools guide:

### Prusti - Ownership Verification

```rust
use prusti_contracts::*;

#[requires(v.len() > 0)]
#[ensures(v.len() > 0)]
fn keep_non_empty(v: &mut Vec<i32>) {
    if v.is_empty() {
        v.push(0);
    }
}
```

### Kani - Bounded Model Checking

```rust
#[cfg(kani)]
#[kani::proof]
fn verify_div() {
    let a: i32 = kani::any();
    let b: i32 = kani::any();
    kani::assume(b != 0);
    let result = a / b;
}
```

### Creusot - Deductive Verification

```rust
use creusot_contracts::*;

#[requires(n <= 12)]
#[ensures(result > 0)]
fn factorial(n: u32) -> u64 {
    if n == 0 { 1 } else { n as u64 * factorial(n - 1) }
}
```
