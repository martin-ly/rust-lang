## 🚀 使用方式

### 基本使用

```toml
[dependencies]
c10_networks = "0.1.0"
```

### 运行示例

```bash
# TCP Echo服务器
cargo run --example tcp_echo_server

# HTTP客户端
cargo run --example http_client

# WebSocket演示
cargo run --example websocket_demo
```

### 运行测试

```bash
# 运行所有测试
cargo test

# 运行特定测试套件
cargo test --test unit_tests
cargo test --test integration_tests
```
