- **引用类型**: 共享引用、可变引用、裸指针
- **高级类型**: GAT、RPITIT、impl Trait

### 2. 泛型系统深度解析

- **类型参数**: 泛型函数、结构体、枚举

- **常量泛型**: 数组长度、类型级计算
- **关联类型**: GAT、RPITIT

- **隐式转换**: Deref强制转换

- **TryFrom/TryInto**: 可失败转换
- **as转换**: 已知安全转换
- **transmute**: 不安全内存重解释
