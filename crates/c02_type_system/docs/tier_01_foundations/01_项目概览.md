## 🎯 项目定位

**Rust 类型系统学习模块** (c02_type_system) 是一个系统化的学习资源，专注于 Rust 语言的核心类型系统，包括基础类型、复合类型、泛型编程、trait 系统和生命周期管理。

本项目旨在帮助学习者：

- ✅ 从零基础到精通 Rust 类型系统
- ✅ 理解类型系统的理论基础
- ✅ 掌握实际开发中的类型应用
- ✅ 建立系统化的知识体系

## 🚀 核心学习内容

### 1. 基础类型系统

- **标量类型**: 整数、浮点数、布尔值、字符
- **复合类型**: 元组、数组、结构体、枚举
- **引用类型**: 借用和引用的类型系统
- **指针类型**: 原始指针和智能指针

**快速开始**: [Tier 2 - 基础类型指南](../tier_02_guides/01_基础类型指南.md)

### 2. 泛型编程

- **泛型函数**: 编写通用的函数
- **泛型结构体**: 定义泛型数据结构
- **泛型枚举**: 创建泛型枚举类型
- **类型约束**: 使用 trait 约束泛型参数

**快速开始**: [Tier 2 - 泛型编程指南](../tier_02_guides/03_泛型编程指南.md)

### 3. Trait 系统

- **Trait 定义**: 定义抽象接口
- **Trait 实现**: 为类型实现 trait
- **Trait 对象**: 使用动态分发
- **关联类型**: 定义类型关联关系

**快速开始**: [Tier 2 - Trait 系统指南](../tier_02_guides/04_Trait系统指南.md)

### 4. 生命周期管理

- **生命周期注解**: 显式标注生命周期
- **生命周期省略**: 理解省略规则
- **生命周期约束**: 使用生命周期约束
- **高级生命周期**: 复杂场景的生命周期

**快速开始**: [Tier 2 - 生命周期指南](../tier_02_guides/05_生命周期指南.md)

**检验标准**:
