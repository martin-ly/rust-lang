## 🎯 核心概念

1. **条件表达式**
   - `if` / `else if` / `else`
   - `if let` 简化模式

2. **循环**
   - `loop` - 无限循环
   - `while` - 条件循环
   - `for` - 迭代循环
   - `while let`

3. **模式匹配**
   - `match` 表达式
   - 解构
   - 守卫子句
   - `@` 绑定

4. **流程控制**
   - `break` / `continue`
   - 标签跳出
   - `return`
