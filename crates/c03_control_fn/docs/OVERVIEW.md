#### 错误处理边界案例（补全）

- From/Into 错误映射； anyhow vs thiserror；早返回与资源释放（RAII）
- 样例：

```rust
fn parse(data: &str) -> Result<u32, std::num::ParseIntError> { data.parse() }

fn process(s: &str) -> anyhow::Result<()> {
    let n = parse(s)?; // 边界：错误类型不一致时用 `map_err` 或 `?` + `From`
    println!("{}", n);
    Ok(())
}
```
