#### Q1: 为什么要升级到 Rust 1.90？

- 编译器性能提升（LLD链接器）
- 新的稳定 API
- const 函数增强
- 更好的 lint 检查

#### Q2: 如何升级到 Rust 1.90？

- 更新 Cargo.toml
- 处理新的 lint 警告
- 测试代码兼容性

#### Q3: Rust 1.90 有哪些重大变化？

- LLD 链接器默认启用
- 新增标准库 API
- const 函数增强
- 生命周期 lint 改进

#### Q4: 如何处理 rust_189_*.rs 文件？

- 这些是历史版本示例
- 保留用于对比学习
- 实际项目使用新版本特性

#### Q5: 迁移过程中常见问题？

- lint 警告解决

### 任务 3: 创建术语表 (20分钟)

**目标**: 创建 Rust 1.90 关键术语表

**术语表内容**:

1. **LLD 链接器 (LLD Linker)**
   - LLVM 的高性能链接器
   - Rust 1.90 在 Linux x86_64 上默认启用
   - 链接速度提升约 2倍

2. **const 函数增强 (Const Function Enhancement)**
   - 更多函数可在 const 上下文中使用
   - 如 `<[T]>::reverse()`、`f32/f64` 数学函数

3. **mismatched_lifetime_syntaxes lint**
   - 检查生命周期语法一致性
   - Rust 1.90 默认启用

4. **Edition 2024**
   - Rust 2024 版次
   - 配合 Rust 1.90 使用

5. **checked_sub_signed()**
   - 新增的带符号减法检查方法
   - 适用于无符号整数类型
