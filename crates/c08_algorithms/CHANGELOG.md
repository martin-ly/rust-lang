## 版本说明

### 版本号格式

我们使用 [语义化版本控制](https://semver.org/lang/zh-CN/)：

- **主版本号**: 不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 版本类型

#### 🚀 重大版本 (Major)

- 破坏性变更
- 不兼容的 API 修改
- 重大架构调整

#### ✨ 功能版本 (Minor)

- 新功能添加
- 向下兼容的功能性新增
- 性能改进

#### 🔧 修复版本 (Patch)

- Bug 修复
- 文档更新
- 小功能改进

## 升级指南

### 从 0.0.1 升级到 0.1.0

#### 破坏性变更

- 需要 Rust 1.89.0 或更高版本
- 部分 API 接口变更
- 异步 API 需要 `async/await` 支持

#### 升级步骤

1. 更新 Rust 到 1.89.0 或更高版本
2. 更新 Cargo.toml 依赖
3. 修改代码以使用新的异步 API
4. 运行测试确保兼容性

#### 兼容性说明

- 同步 API 保持向后兼容
- 异步 API 需要代码重构
- 性能优化自动生效
