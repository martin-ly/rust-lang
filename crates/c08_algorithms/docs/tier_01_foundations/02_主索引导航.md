## 🗺️ 学习路径导航

### 🌱 新手路径 (4-6周)

**Week 1-2: 基础算法**:

```text
第1周
├─ 排序算法基础 (冒泡、选择、插入)
├─ 时间复杂度入门
└─ Rust基础回顾

第2周
├─ 高级排序 (快排、归并、堆排)
├─ 搜索算法 (线性、二分)
└─ 实践练习
```

**Week 3-4: 数据结构**:

```text
第3周
├─ 线性表 (数组、链表、栈、队列)
├─ 基础树结构 (二叉树、BST)
└─ 图的表示 (邻接表、邻接矩阵)

第4周
├─ 树的遍历 (前序、中序、后序、层序)
├─ 图的遍历 (BFS、DFS)
└─ 综合练习
```

**Week 5-6: 进阶算法**:

```text
第5周
├─ 动态规划入门 (斐波那契、爬楼梯)
├─ 贪心算法基础
└─ 回溯算法简介

第6周
├─ 综合项目实战
├─ 算法竞赛练习
└─ 总结复习
```

---

**Week 1: 核心算法**:

**Week 2: 高级主题**:
