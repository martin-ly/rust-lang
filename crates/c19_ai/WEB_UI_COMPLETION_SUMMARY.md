## 🚀 使用方法

### 启动Web UI服务器

```bash
# 运行Web UI示例
cargo run --example web_ui_server

# 或者使用完整功能
cargo run --features "api-server" --example web_ui_server
```

### 访问管理界面

打开浏览器访问: <http://localhost:8080>
