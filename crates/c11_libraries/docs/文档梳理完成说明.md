## 📚 如何使用新文档

1. 从 **[文档中心](README.md)** 开始
2. 查看 **[快速导航](00_MASTER_INDEX.md)** 的初学者路径
3. 阅读 [guides/sql.md](guides/sql.md) 和 [guides/redis.md](guides/redis.md)
