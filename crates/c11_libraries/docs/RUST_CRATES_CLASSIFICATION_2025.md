## 分类方法论

### 多维分类体系

```text
Rust 库分类维度
├─ 功能维度 (What)
│  ├─ 基础设施 (序列化、时间、随机数)
│  ├─ 系统编程 (并发、内存、I/O)
│  ├─ 应用开发 (Web、数据库、网络)
│  ├─ 领域特定 (游戏、GUI、嵌入式)
│  └─ 工具链 (测试、文档、部署)
├─ 场景维度 (Where)
│  ├─ 后端服务
│  ├─ 前端应用
│  ├─ CLI 工具
│  ├─ 嵌入式系统
│  └─ WebAssembly
├─ 成熟度维度 (How Stable)
│  ├─ 生产就绪 (⭐⭐⭐⭐⭐)
│  ├─ 稳定可用 (⭐⭐⭐⭐)
│  ├─ 快速发展 (⭐⭐⭐)
│  └─ 实验性 (⭐⭐)
└─ 依赖维度 (Dependency)
   ├─ 零依赖
   ├─ 轻量依赖
   └─ 重度依赖
```

```toml
[dependencies]

clap = { version = "4", features = ["derive"] }

```
