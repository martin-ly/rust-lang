## 🎯 项目概述

### 目标

创建一个**全面、深入、系统化**的 Rust 必备开源库知识体系，采用5层架构分类法，覆盖从基础设施到工具链的完整开发栈。

### 设计原则

1. **多维分类**: 功能、场景、成熟度三维组织
2. **深度与广度**: 107+ 核心库，200+ 代码示例
3. **实用导向**: 可运行代码、场景模板、最佳实践
4. **持续更新**: 基于 Rust 1.90 最新状态
