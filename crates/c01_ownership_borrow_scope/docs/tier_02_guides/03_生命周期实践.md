### 2. 生命周期标注

**语法**:

```rust
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() { x } else { y }
}
```

**详细学习**: [生命周期注解完整文档](../02_core/03_lifetime_annotations.md)

### 3. 生命周期省略规则

**三大规则**:

1. 每个引用参数获得独立的生命周期
2. 只有一个输入生命周期时，输出使用它
3. 方法中，输出使用 `&self` 的生命周期

**详细学习**: [生命周期注解完整文档](../02_core/03_lifetime_annotations.md)
