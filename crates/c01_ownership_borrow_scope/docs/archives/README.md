## 📝 归档策略

### 什么时候归档

文档会在以下情况下被归档：

1. **版本升级**: Rust 版本更新导致内容过时
2. **结构重组**: 文档体系重新组织
3. **内容整合**: 多个文档合并为一个
4. **重复冗余**: 与其他文档内容重复
5. **历史遗留**: 项目演进中产生的临时文档

### 归档原则

- ✅ **保留所有内容**: 不删除，只归档
- ✅ **明确分类**: 清晰的子目录结构
- ✅ **提供说明**: 解释归档原因和替代方案
- ✅ **可追溯**: 保持文件的Git历史
