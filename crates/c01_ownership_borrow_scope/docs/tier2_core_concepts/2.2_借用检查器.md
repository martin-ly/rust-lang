## 🎯 核心内容导航

### 1. 借用规则

**核心规则**:

```text
✅ 任意数量的不可变借用 (&T)
   OR
✅ 有且仅有一个可变借用 (&mut T)

✅ 借用必须总是有效（不能悬垂）
```

**详细学习**: [借用系统完整文档](../02_core/02_borrowing_system.md)

### 2. 借用检查器机制

**关键概念**:

- 编译时静态分析
- 借用作用域检查
- NLL (Non-Lexical Lifetimes) 改进

**详细学习**: [借用系统完整文档](../02_core/02_borrowing_system.md)

### 3. 不可变借用与可变借用

**对比**:

| 特性 | 不可变借用 `&T` | 可变借用 `&mut T` |
|------|----------------|-------------------|
| 数量 | 无限制 | 唯一 |
| 并发 | 允许 | 互斥 |
| 修改 | 不可 | 可以 |

**详细学习**: [借用系统完整文档](../02_core/02_borrowing_system.md)

## 💡 快速参考

### 借用规则速查

```rust
let s = String::from("hello");

// ✅ 多个不可变借用
let r1 = &s;
let r2 = &s;

// ✅ 一个可变借用（不可变借用已结束）
let mut s = String::from("hello");
let r = &mut s;

// ❌ 错误：同时有不可变和可变借用
let r1 = &s;
let r2 = &mut s;  // 编译错误
```
