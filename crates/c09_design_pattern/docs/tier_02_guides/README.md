## 🔗 模式分类

**创建型模式** (Creational):

- 单例、工厂、建造者、原型

**结构型模式** (Structural):

- 适配器、装饰器、代理、外观

**行为型模式** (Behavioral):

- 观察者、策略、命令、状态

**并发模式** (Concurrency):

- Actor、Reactor、CSP、Future/Promise
