# 入门指南总览

本目录汇集安装与快速上手内容，帮助你在 10 分钟内运行首个模型。

## 导航

- 安装与环境准备: 请见 `installation.md`
- 快速开始示例: 请见 `quick-start.md`

## 你将获得

- 成功构建并运行 `MM1Queue`、`LinearRegression`、`FiniteStateMachine` 的最小示例
- 运行测试/基准的常用命令速查

## 先决条件

- Rust 1.70+，推荐使用最新稳定版
- 已安装 `cargo`

## 常见问题 (FAQ)

- 如遇依赖解析失败，先运行 `cargo update` 再重试
- Windows/WSL/Linux/macOS 均已验证可用
