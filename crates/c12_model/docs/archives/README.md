## 📝 归档原则

### 归档标准

文档被归档的原因：

1. **内容过时** - 描述的功能已被更新
2. **临时性质** - 特定时期的临时文档
3. **已完成** - 描述的任务已完成
4. **历史记录** - 纯粹的历史记录文档

### 不应归档的文档

- 当前有效的技术文档
- API 参考文档
- 使用指南和教程
- 核心概念说明
