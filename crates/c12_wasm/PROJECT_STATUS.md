## 🚀 如何使用

### 快速开始

```bash
# 1. 编译库
cargo build --lib

# 2. 运行测试
cargo test

# 3. 构建 WASM 模块
wasm-pack build --target web

# 4. 启动演示
python -m http.server 8080
# 访问: http://localhost:8080/demo/
```

### 学习路径

1. **第1周**: 阅读 Tier 1 文档，运行基础示例
2. **第2周**: 学习 Tier 2 文档，实践 JavaScript 互操作
3. **第3-4周**: 深入 Tier 3 和 Tier 4，学习高级主题
4. **第5-8周**: 创建自己的 WASM 项目
