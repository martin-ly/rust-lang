## 🎯 项目简介

### 核心定位

**C12 WASM** 是最全面的 Rust WebAssembly 学习资源，专注于：

- **✅ WASM 核心概念**: 二进制格式、线性内存、指令集、执行模型
  - 栈式虚拟机、函数调用、内存管理
  - WAT 文本格式、模块系统、导入导出

- **✅ Rust 编译 WASM**: 完整的工具链和编译流程
  - rustc + wasm32-unknown-unknown 目标
  - wasm-bindgen 互操作工具
  - wasm-pack 包管理和发布

- **✅ JavaScript 集成**: 前端和后端集成方案
  - React/Vue/Angular 框架集成
  - Node.js 运行时集成
  - TypeScript 类型定义

- **✅ 性能优化**: 二进制大小和运行时性能
  - 编译优化选项
  - wasm-opt 工具链
  - 内存管理最佳实践

- **✅ 实战应用**: 真实世界应用场景
  - 前端应用、游戏开发、科学计算
  - 边缘计算、命令行工具、全栈应用

### 为什么学习 WASM？

**核心价值**:

1. **✅ 高性能执行**
   - 接近原生代码的执行速度
   - 比 JavaScript 快 2-10 倍
   - 适合性能敏感场景

2. **✅ 跨平台能力**
   - 一次编写，到处运行
   - 浏览器、服务器、边缘设备
   - 统一的运行时环境

3. **✅ 安全性保障**
   - 沙箱执行环境
   - 内存安全保证
   - 类型安全验证

4. **✅ 生态系统**
   - 丰富的工具链支持
   - 活跃的社区生态
   - 广泛的应用场景

5. **✅ 职业发展**
   - 全栈开发的必备技能
   - 性能优化的核心工具
   - 未来技术趋势

### Rust 在 WASM 中的优势

**为什么用 Rust 开发 WASM？**

1. **✅ 内存安全**
   - 编译时保证内存安全
   - 无需运行时 GC
   - 零成本抽象

2. **✅ 性能优秀**
   - 编译器优化能力强
   - 生成的 WASM 代码高效
   - 二进制体积可控

3. **✅ 类型系统**
   - 强类型系统
   - 丰富的类型映射
   - 编译时错误检查

4. **✅ 工具链完善**
   - wasm-bindgen 自动化工具
   - wasm-pack 完整工作流
   - 丰富的文档和示例

5. **✅ 社区支持**
   - 官方支持良好
   - 活跃的社区生态
   - 丰富的学习资源

## 🚀 快速开始

### 环境准备

```bash
# 1. 安装 Rust 工具链 (如果尚未安装)
rustup update stable

# 2. 添加 WASM 编译目标
rustup target add wasm32-unknown-unknown

# 3. 安装 wasm-pack
curl https://rustwasm.github.io/wasm-pack/installer/init.sh -sSf | sh

# 4. 验证安装
rustc --version  # 应该显示 1.90+
wasm-pack --version
```

### 第一个 WASM 项目

```bash
# 1. 创建新项目
wasm-pack new hello-wasm
cd hello-wasm

# 2. 构建 WASM 模块
wasm-pack build

# 3. 在浏览器中测试
cd www
npm install
npm start
```

### 运行示例

```bash
# 运行所有示例
cargo run --example hello_wasm

# 运行特定示例
cargo run --example canvas_demo

# 运行测试
cargo test

# 构建 WASM
wasm-pack build --target web
```
