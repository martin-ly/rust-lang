# Rust 交互式学习测验

欢迎来到 Rust 交互式学习测验系统！这是一个基于 [mdbook-quiz](https://github.com/willcrichton/mdbook-quiz) 的交互式测验平台，帮助你巩固 Rust 编程的核心概念。

## 🎯 学习目标

通过本测验系统，你将能够：

- 检验对 Rust 所有权系统的理解
- 掌握借用检查器的工作机制
- 深入理解类型系统的强大功能
- 学会处理复杂的生命周期问题
- 掌握异步编程的核心概念

## 🎮 如何使用

1. 点击上方的测验链接进入对应主题
2. 阅读题目并选择你认为正确的答案
3. 提交后查看答案解析
4. 每道题都包含详细的解释说明

## ⚡ 测验题型

- **单选题**：从多个选项中选择唯一正确答案
- **多选题**：可能有多个正确答案，需要全部选对
- **判断题**：判断陈述的正误
- **代码填空**：补全代码使其能通过编译

## 🏆 评分标准

- **基础题**：每题 5 分，共 60 分
- **进阶题**：每题 10 分，共 40 分
- **挑战题**：每题 15 分，共 45 分

**满分**：145 分
