## 📚 学习路径

### 第一部分：Rust 基础 (2-4 周)

打下坚实的基础，理解 Rust 的核心概念。

- **[C01: 所有权与借用](./c01_ownership/README.md)** ⭐⭐⭐⭐⭐
  - Rust 最重要的特性
  - 所有权、借用、生命周期
  - 内存安全保证

- **[C02: 类型系统](./c02_type_system/README.md)** ⭐⭐⭐⭐⭐
  - Trait 系统
  - 泛型编程
  - 类型安全

- **[C03: 控制流与函数](./c03_control_fn/README.md)** ⭐⭐⭐⭐
  - 条件控制与模式匹配
  - 循环与迭代器
  - 函数与闭包

### 第二部分：并发与异步 (4-8 周)

掌握 Rust 的高级并发特性。

- **[C04: 泛型编程](./c04_generic/README.md)** ⭐⭐⭐⭐
  - 关联类型
  - 高阶 Trait
  - GATs (泛型关联类型)

- **[C05: 线程与并发](./c05_threads/README.md)** ⭐⭐⭐⭐
  - 线程创建与管理
  - 共享状态并发
  - 消息传递
  - 原子操作

- **[C06: 异步编程](./c06_async/README.md)** ⭐⭐⭐⭐⭐
  - async/await 语法
  - Future 与 Stream
  - 异步 Runtime
  - Tokio 生态

### 第三部分：系统与应用 (8-12 周)

学习系统编程和实际应用开发。

- **[C07: 进程管理](./c07_process/README.md)** ⭐⭐⭐
  - 进程创建与管理
  - 进程间通信 (IPC)
  - 信号处理

- **[C08: 算法与数据结构](./c08_algorithms/README.md)** ⭐⭐⭐
  - 经典算法实现
  - 高性能数据结构
  - 算法优化技巧

- **[C09: 设计模式](./c09_design_pattern/README.md)** ⭐⭐⭐⭐
  - GoF 经典模式
  - Rust 特定模式
  - 最佳实践

- **[C10: 网络编程](./c10_networks/README.md)** ⭐⭐⭐⭐
  - TCP/UDP 编程
  - HTTP 服务
  - WebSocket
  - gRPC

### 第四部分：生产实践 (持续学习)

深入高级特性和生产级实践。

- **[C11: 宏系统](./c11_macro/README.md)** ⭐⭐⭐
  - 声明宏
  - 过程宏
  - 元编程

## 📊 学习建议

### 新手必读

1. **不要跳过基础**：C01-C03 是 Rust 的基石，必须扎实掌握
2. **多写代码**：每个概念都要自己动手实践
3. **理解错误信息**：Rust 编译器的错误提示非常有价值
4. **循序渐进**：遇到困难是正常的，慢慢来

### 进阶技巧

1. **对比学习**：将 Rust 与你熟悉的语言对比
2. **阅读源码**：查看标准库和优秀项目的实现
3. **参与社区**：加入 Rust 中文社区讨论
4. **实战项目**：完成附录中的实战项目
