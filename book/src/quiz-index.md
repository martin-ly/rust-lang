# 测验索引

本页面汇总了所有可用的 Rust 交互式测验。

## 核心概念测验

### 1. 所有权系统

所有权是 Rust 最核心的概念，理解所有权是掌握 Rust 的关键。

- 📄 [所有权测验](./quizzes/ownership.md)
- 📝 12 道题目
- 🎯 涵盖：Move 语义、Copy trait、Clone trait、Drop trait、所有权转移

### 2. 借用检查器

借用检查器是 Rust 保证内存安全的核心机制。

- 📄 [借用检查器测验](./quizzes/borrowing.md)
- 📝 12 道题目
- 🎯 涵盖：不可变引用、可变引用、引用规则、悬垂引用、内部可变性

### 3. 类型系统

Rust 的类型系统提供了强大的抽象能力和编译时安全检查。

- 📄 [类型系统测验](./quizzes/types.md)
- 📝 12 道题目
- 🎯 涵盖：类型推导、泛型、Trait、关联类型、newtype 模式

### 4. 生命周期

生命周期确保引用始终指向有效的数据。

- 📄 [生命周期测验](./quizzes/lifetimes.md)
- 📝 12 道题目
- 🎯 涵盖：生命周期标注、生命周期省略规则、'static、生命周期边界

### 5. 异步编程

异步编程允许你编写高效的并发代码。

- 📄 [异步编程测验](./quizzes/async.md)
- 📝 12 道题目
- 🎯 涵盖：async/await、Future、Pin、async trait、运行时选择

## 难度说明

| 难度级别 | 说明 | 建议完成时间 |
|----------|------|--------------|
| 🟢 基础 | 考察基本概念，适合初学者 | 每题 1-2 分钟 |
| 🟡 进阶 | 需要理解深层原理 | 每题 3-5 分钟 |
| 🔴 挑战 | 涉及复杂场景和边界情况 | 每题 5-10 分钟 |

## 学习建议

1. **按顺序学习**：建议按照所有权 → 借用 → 类型 → 生命周期 → 异步的顺序进行
2. **理解错误**：不要只关注正确答案，更要理解错误选项为什么错
3. **动手实验**：将代码复制到本地运行，加深理解
4. **复习巩固**：定期回顾错题，强化记忆
