## 🎯 统一标准

### 1. 版本信息标准

#### Rust 版本

- **统一格式**: `Rust 1.91.1` 或 `Rust 1.91.1+`
- **禁止使用**: `Rust 1.89`, `Rust 1.90`, `Rust 1.91`（不完整）
- **文档头部格式**:

  ```markdown
  **最后更新**: 2025-11-15
  **适用版本**: Rust 1.91.1+
  **Rust版本**: 1.91.1
  ```

#### 依赖版本

- **必须与 Cargo.toml 一致**
- **主要依赖版本**（2025-11-15）:
  - tokio: 1.48.0

  - axum: 0.8.7

  - bytes: 1.11.0
  - config: 0.15.19

  - leptos: 0.8.12

  - egui: 0.33.2

### 2. 日期信息标准

#### 统一格式

- **最后更新**: `2025-11-15`
- **制定日期**: `2025-11-15`
- **文档日期**: `2025年11月15日`
- **禁止使用**: `2025-10-22`, `2025-10-30`, `2025-01-27` 等旧日期

### 3. 目录结构标准

#### 统一序号格式

```markdown
## 1. 一级标题
### 1.1 二级标题
#### 1.1.1 三级标题
##### 1.1.1.1 四级标题（少用）
```

#### 文档分类标准

```markdown
## 📋 目录

### 1. 基础概念
#### 1.1 核心概念
#### 1.2 基础语法

### 2. 进阶主题
#### 2.1 高级特性
#### 2.2 最佳实践

### 3. 参考文档
#### 3.1 API 参考
#### 3.2 术语表
```

### 4. 语义一致性标准

#### 术语统一

- **所有权系统**: 统一使用"所有权系统"，不使用"所有权机制"
- **借用检查器**: 统一使用"借用检查器"，不使用"借用检查"
- **生命周期**: 统一使用"生命周期"，不使用"生存期"
- **异步编程**: 统一使用"异步编程"，不使用"异步开发"

#### 版本特性描述

- **Rust 1.91.1**: 描述为"最新稳定版本"或"当前稳定版本"
- **特性说明**: 必须标注"Rust 1.91.1 新特性"或"Rust 1.91.1 改进"
