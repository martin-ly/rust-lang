3. **消息传递理论**：提出了基于消息传递的并发模型，包括通道、发送者和接收者

5. **原子操作理论**：设计了原子操作系统的数学模型，包括原子性、一致性和隔离性
6. **并发安全理论**：建立了并发安全的形式化定义，包括数据竞争、死锁和活锁的检测与预防
7. **异步系统理论**：提出了异步编程的形式化模型，包括Future、async/await和异步运行时
