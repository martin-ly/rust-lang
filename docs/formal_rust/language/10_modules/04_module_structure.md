## 1. 文件系统映射策略

- 文件/目录结构与模块树一一对应
- mod.rs用于目录模块，单文件可内联mod

## 2. 内联模块与外部模块

- 内联：mod foo { ... }
- 外部：mod foo; // 对应foo.rs或foo/mod.rs

## 3. 工程案例

```rust
// src/lib.rs
mod a; // 对应src/a.rs
mod b { pub fn bar() {} } // 内联模块
```

## 4. 批判性分析与未来展望

- 文件系统映射提升可维护性，但多层嵌套时易混淆
- 未来可探索自动化模块结构可视化与重构工具
