## 2. 工程案例

```rust
// O(n)线性复杂度示例
fn linear_search(arr: &[i32], target: i32) -> Option<usize> {
    for (i, &x) in arr.iter().enumerate() {
        if x == target { return Some(i); }
    }
    None
}
```
