## 模块概述

类型检查语义模块是Rust类型系统语义的重要组成部分，涵盖了类型检查的完整语义定义，包括检查规则、错误处理、类型安全和检查优化等核心概念。本模块建立了严格的理论基础，为Rust语言的类型检查系统提供了形式化的语义定义。
