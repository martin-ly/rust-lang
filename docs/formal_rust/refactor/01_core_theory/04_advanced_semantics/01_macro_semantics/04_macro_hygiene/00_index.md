## 模块概述

宏卫生语义模块是Rust宏系统语义的关键组成部分，涵盖了宏卫生的完整语义定义，包括变量卫生、标识符卫生、作用域卫生和卫生保证等核心概念。本模块建立了严格的理论基础，为Rust语言的宏卫生系统提供了形式化的语义定义。
