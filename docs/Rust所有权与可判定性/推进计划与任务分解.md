## 📋 项目概述

### 目标

建立Rust所有权系统的全面形式语义分析框架，服务于：

- 学术研究人员（形式化方法、类型系统）
- 高级工程师（理解编译器行为、优化代码）
- Rust学习者（深入理解所有权系统）
