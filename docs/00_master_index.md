# 项目总索引 {#项目总索引}

> **Rust 版本**: 1.96.0+ (Edition 2024)

本项目是一个系统化的 Rust 学习资源库，覆盖从基础到形式化验证的完整知识谱系。

## 快速入口 {#快速入口}

| 入口 | 路径 | 说明 |
|:---|:---|:---|

| **概念文件** | [concept/](../concept) | L1-L7 分层概念文档 |
| **知识索引** | [knowledge/](../knowledge) | 系统化知识卡片 |

| **Crate 示例** | [crates/](../crates) | 可编译代码示例 |

