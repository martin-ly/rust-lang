## 形式化定义规范

每个模式包含以下形式化要素：

```
Def [模式名]  :=  核心概念定义
Axiom [A编号]  :=  基本假设
Theorem [T编号]  :=  可证明的性质
Proof  :=  L2 级自然语言证明
```
