> **最后更新**: 2026-03-08

## 概述

分布式系统模式的对比矩阵。

## 模式矩阵

| 模式 | 一致性 | 可用性 | 分区容错 | 复杂度 |
|------|--------|--------|----------|--------|
| 两阶段提交 | 强 | 低 | 否 | 中 |
| Saga | 最终 | 高 | 是 | 中 |
| CQRS | 最终 | 高 | 是 | 高 |
| 事件溯源 | 最终 | 高 | 是 | 高 |

## 选型建议

- 强一致性需求: 2PC/Paxos
- 高可用需求: Saga + 事件溯源
- 简单场景: 主从复制
