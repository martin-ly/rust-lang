## 工具选择建议

| 场景 | 推荐工具 | 理由 |
| :--- | :--- | :--- |
| 日常开发检查 | Miri | 内置，易用 |
| CI自动化 | Kani | 自动化属性验证 |
| 合约编程 | Prusti | 有前置/后置条件 |
| 函数式证明 | Creusot | 纯函数友好 |
| 系统代码 | Verus | 线性类型支持 |
| 研究/教学 | RustBelt | 理论基础 |
| Coq/Lean 对接 | 已归档 | 见 [archive/deprecated/](../../archive/deprecated/)；主路径聚焦 Prusti/Kani |

**主推路径**：Prusti、Kani（Rust 原生，无需专业形式化语言）。Coq/Lean 为可选进阶。
