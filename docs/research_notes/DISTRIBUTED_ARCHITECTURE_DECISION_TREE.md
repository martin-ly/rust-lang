# 分布式架构决策树

> **最后更新**: 2026-03-08

---

## 概述

帮助选择适合的分布式架构模式。

## 决策流程

```
需要强一致性?
├── 是 -> 需要高可用?
│   ├── 是 -> Paxos/Raft
│   └── 否 -> 2PC/3PC
└── 否 -> 需要分区容错?
    ├── 是 -> 最终一致性 (Gossip)
    └── 否 -> 主从复制
```

## 架构模式

| 场景 | 推荐模式 | 说明 |
|------|----------|------|
| 微服务 | 服务网格 | 可观测性 |
| 大数据 | 流处理 | Kafka |
| 事务 | Saga | 补偿事务 |
