## 🌳 3. 决策树图 (Decision Trees)

### 3.1 所有权选择决策树

```mermaid
graph TD
    A[需要共享数据?] -->|是| B[需要修改?]
    A -->|否| C[所有权转移<br/>move]

    B -->|是| D[编译期确定生命周期?]
    B -->|否| E[不可变借用 &T]

    D -->|是| F[可变借用 &mut T]
    D -->|否| G[运行时检查?]

    G -->|是| H[RefCell<T>]
    G -->|否| I[线程安全?]

    I -->|是| J[Mutex<T> 或 RwLock<T>]
    I -->|否| K[RefCell<T>]

    F --> L[多个所有者?]
    L -->|是| M[线程安全?]
    M -->|是| N[Arc<Mutex<T>>]
    M -->|否| O[Rc<RefCell<T>>]
    L -->|否| P[单一可变借用 &mut T]

    E --> Q[多个所有者?]
    Q -->|是| R[线程安全?]
    R -->|是| S[Arc<T>]
    R -->|否| T[Rc<T>]
    Q -->|否| U[单一不可变借用 &T]
```

### 3.2 错误处理策略决策树

```mermaid
graph TD
    A[遇到错误情况?] --> B[错误可恢复?]

    B -->|否| C[panic!<br/>不可恢复]
    B -->|是| D[需要错误详情?]

    D -->|否| E[Option<T><br/>Some/None]
    D -->|是| F[需要传播?]

    F -->|否| G[Result<T,E><br/>match 处理]
    F -->|是| H[在迭代中?]

    H -->|是| I[需要提前终止?]
    H -->|否| J[Result + ? 操作符]

    I -->|是| K[ControlFlow<br/>Break/Continue]
    I -->|否| L[Result + try_fold<br/>collect]
```

### 3.3 并发策略决策树

```mermaid
graph TD
    A[需要并发?] --> B[共享数据?]

    B -->|否| C[消息传递<br/>Channel]
    B -->|是| D[只读共享?]

    D -->|是| E[Arc<T><br/>T: Sync]
    D -->|否| F[读写分离?]

    F -->|是| G[RwLock<T><br/>读多写少]
    F -->|否| H[Mutex<T><br/>通用保护]

    C --> I[多生产者?]
    I -->|是| J[mpsc::channel]
    I -->|否| K[oneshot channel]

    E --> L[需要修改?]
    L -->|是| M[内部可变性?<br/>Mutex/RwLock]

```

### 3.4 泛型边界决策树

```mermaid
graph TD
    A[定义泛型参数?] --> B[需要比较?]

    B -->|是| C[需要排序?]
    B -->|否| D[需要克隆?]

    C -->|是| E[T: Ord]
    C -->|否| F[T: PartialEq]

    D -->|是| G[需要深拷贝?]
    D -->|否| H[需要打印?]

    G -->|是| I[T: Clone]
    G -->|否| J[T: Copy]

    H -->|是| K[调试格式?<br/>T: Debug]

    F --> M
    I --> M
```

### 3.5 Rust 1.94 迭代器选择决策树

```mermaid
graph TD

    C -->|是| E[Rust 1.94<br/>array_windows::<N>]

    E --> G[需要累积?]
```
