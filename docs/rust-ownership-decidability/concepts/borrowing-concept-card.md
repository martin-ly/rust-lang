# 借用概念卡片

## 概念定义

**借用(Borrowing)**: 临时获取值的引用而不转移所有权。

形式化:

```
Borrow(v, T, k) ≡ 获得类型T的值v的引用，借用类型为k
其中 k ∈ {Shared, Mutable}
```

## 借用规则

```
规则: 在任意时刻，只能满足以下条件之一:
  1. 一个可变引用 (&mut T)
  2. 任意数量的不可变引用 (&T)
```

## 借用生命周期

```
代码示例:
  let x = 5;
  let r = &x;        // 借用开始
  println!("{}", r); // 使用借用
  // r 在这里结束   // 借用结束
```

## 反例: 违反借用规则

```rust
let mut s = String::from("hello");
let r1 = &s;
let r2 = &mut s;  // 错误! 不能同时有&和&mut
println!("{}", r1);
```

## 场景决策

| 需求 | 使用 | 原因 |
|------|------|------|
| 只读访问 | &T | 允许多个读者 |
| 修改数据 | &mut T | 独占访问保证安全 |
| 临时访问 | 借用 | 不转移所有权 |
