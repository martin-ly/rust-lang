# 生命周期概念卡片

## 概念定义

**生命周期(Lifetime)**: 引用有效的代码区域。

形式化:

```
Lifetime('a) ≡ 控制流图中引用有效的点集
'a ⊆ CFG_Points
```

## 核心关系

```
生命周期包含关系: 'a: 'b  ⟺  'b ⊆ 'a

'a (较长)  ┌─────────────────────┐
'b (较短)      ┌───────────┐
                └───────────┘
          └─────────────────────┘
```

## 生命周期省略规则 (ELision)

| 情况 | 输入 | 输出 | 推断规则 |
|------|------|------|---------|
| 单个输入 | fn foo(&T) | &T | 输入=输出 |
| 多个输入 | fn foo(&T, &U) | 必须标注 | 无法推断 |
| 方法 | fn foo(&self) | &self | 方法接收者生命周期 |

## 常见错误

```rust
// 错误: 返回局部变量引用
fn bad() -> &String {
    let s = String::from("x");
    &s  // s在这里被释放
}

// 正确: 返回所有权
fn good() -> String {
    String::from("x")
}
```

## NLL (Non-Lexical Lifetimes)

```rust
let mut x = vec![1, 2, 3];
let y = &x[0];      // 借用开始
println!("{}", y);  // 借用结束 (NLL)
x.push(4);          // OK!
```
