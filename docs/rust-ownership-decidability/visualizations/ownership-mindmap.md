```mermaid
graph TD
    A[所有权 Ownership] --> B[核心规则]
    A --> C[操作类型]

    A --> E[编译时检查]

    B --> B1[唯一性: 单所有者]

    C2 --> C2a[不可变借用 &T]
    C2 --> C2b[可变借用 &mut T]

    D --> D1[线性逻辑]
    D --> D2[仿射逻辑]
    D --> D3[分离逻辑]
    D --> D4[区域理论]

    E --> E1[借用检查器]
    E --> E2[生命周期检查]
    E --> E3[Send/Sync检查]
```
