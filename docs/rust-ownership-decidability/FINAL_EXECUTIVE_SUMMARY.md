## 5 个核心定理

### ✅ 定理 1: Borrow Checking 终止性

对于任何 Linearizable 的类型环境，borrow checking 必然终止。

### ✅ 定理 2: 类型保持

求值保持类型，良类型程序的求值结果仍是良类型的。

### ✅ 定理 3: 进展

良类型的表达式要么已是值，要么可以进一步求值。

### ✅ 定理 4: 类型安全

Preservation + Progress = Type Safety

### ✅ 定理 5: 可判定性

Rust 所有权系统的类型检查是完全可判定的。
