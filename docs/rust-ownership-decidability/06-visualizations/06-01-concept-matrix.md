| 维度 | 线性类型 (Linear) | 仿射类型 (Affine) | 相关类型 (Relevant) | 有序类型 (Ordered) |

| **使用次数** | 恰好一次 | 最多一次 | 至少一次 | 顺序敏感 |
| **代表语言** | Alms, Clean | **Rust**, Vault | 某些λ演算 | Lambdacalc |
| **可判定性** | 不可判定 | **可判定** | 可变 | 可变 |
| **复杂度** | 不可判定 | TOWER | PSPACE | PSPACE |

## 4. 所有权语义多维映射

### 4.1 所有权操作与数学结构

| 操作 | Rust语法 | 范畴论语义 | 分离逻辑 | 线性逻辑 |
|------|---------|-----------|---------|---------|
| 创建 | `let x = v` | 引入对象 | `x ↦ v` | 公理 |
| 移动 | `let y = x` | 态射复合 | 资源转移 | ⊸消除 |
| 复制 | `let y = x` (Copy) | 对角态射 | 持久断言 | !消除 |
| 借用 | `&x`, `&mut x` | 限制标架 | borrow命题 | 模态 |
| 释放 | drop(x) | 终结子 | 释放权限 | 对偶 |

## 5. 并发安全矩阵

### 5.1 Send与Sync的组合矩阵

| | Sync: Yes | Sync: No |
|---|-----------|----------|
| **Send: Yes** | i32, String, Arc<Mutex<T>> | MutexGuard<T> |
| **Send: No** | ! (不可能) | Rc<T>, RefCell<T> |

### 5.2 线程安全与所有权

```text
┌─────────────────────────────────────────────────────────────────────────────┐
│                    并发安全与所有权关系                                       │
├─────────────────────────┬───────────────────────────────────────────────────┤
│      类型/操作          │           线程安全语义                             │
├─────────────────────────┼───────────────────────────────────────────────────┤
│  T: Send                │  所有权可跨线程转移                                │
│                         │  let t = thread::spawn(move || { ... })          │
├─────────────────────────┼───────────────────────────────────────────────────┤
│  T: Sync                │  &T 可跨线程共享                                   │
│                         │  多个线程可同时持有 &T                            │
├─────────────────────────┼───────────────────────────────────────────────────┤
│  T: Send + !Sync        │  可转移但不能共享                                  │
│                         │  例: MutexGuard                                    │
├─────────────────────────┼───────────────────────────────────────────────────┤
│  T: !Send + Sync        │  不可能! (Sync要求 &T: Send)                      │
├─────────────────────────┼───────────────────────────────────────────────────┤
│  &mut T: Send           │  可变引用可转移                                    │
│  (当 T: Send)           │  保证同一时间只有一个使用者                        │
├─────────────────────────┼───────────────────────────────────────────────────┤
│  &T: Send               │  共享引用可转移                                    │
│  (当 T: Sync)           │  允许多个读者                                      │
└─────────────────────────┴───────────────────────────────────────────────────┘
```

## 6. 错误类型与检查阶段矩阵

| 错误类型 | 检查阶段 | 检测方法 | 示例 |
|---------|---------|---------|------|
| Use after move | 借用检查 | 流敏感分析 | `let y = x; use(x)` |
| Double borrow mut | 借用检查 | 别名分析 | `&mut x` 同时存在 |
| Dangling reference | 生命周期检查 | 区域检查 | 返回局部变量引用 |
| Data race | 类型系统 | Send/Sync | 多线程写共享数据 |
| Use after free | 借用检查 | 所有权 | unsafe代码错误 |
