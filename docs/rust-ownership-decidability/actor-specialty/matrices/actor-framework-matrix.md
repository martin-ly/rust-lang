## 快速选择决策表

| 如果你的需求是... | 选择框架 | 理由 |
|:---|:---|:---|
| 快速构建Web应用 | **Actix** | 与Actix-web无缝集成，生态完善 |
| 高可用生产系统 | **Bastion** | 强大的容错和监督树 |
| 分布式微服务 | **coerce** | 原生分布式和集群支持 |
| 学习Actor模型 | **Xtra** | 简单API，易于理解 |
| 极致性能 | **Heph** | 优化的消息传递 |
| 资源受限环境 | **Stakker** | 轻量级，低开销 |
