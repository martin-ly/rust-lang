## 监督树结构图

```text
监督树层次结构:

                    ┌─────────────┐
                    │  Root       │
                    │ Supervisor  │
                    │  (策略:     │
                    │   OneForOne)│
                    └──────┬──────┘
                           │
           ┌───────────────┼───────────────┐
           │               │               │
    ┌──────┴──────┐ ┌──────┴──────┐ ┌──────┴──────┐
    │ Supervisor  │ │ Supervisor  │ │ Worker      │
    │  (Type A)   │ │  (Type B)   │ │  (Leaf)     │
    │ OneForAll   │ │ RestForOne  │ │             │
    └──────┬──────┘ └──────┬──────┘ └─────────────┘
           │               │
      ┌────┴────┐     ┌────┴────┐
      │ Worker  │     │ Worker  │
      │ Actor   │     │ Actor   │
      └─────────┘     └─────────┘

监督策略:
├── OneForOne:  一个失败 → 只重启它
├── OneForAll:  一个失败 → 重启所有兄弟
└── RestForOne: 一个失败 → 重启它和之后启动的
```
