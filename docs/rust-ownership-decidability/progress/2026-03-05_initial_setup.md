## 关键发现

### 1. 可判定性问题的现状

**Featherweight Rust 发现的问题**:

- Borrow checking 在类型左值时可能不终止
- 根本原因是借用可能包含其他左值，导致无限递归

**解决方案**:

- Payet et al. 提出了 `Linearizable` 条件
- 基于类型秩 (rank) 的度量函数可以证明终止性
