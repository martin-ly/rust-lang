## 2. 类型级数字

### 定理 2.1 (Peano表示)

> 数字编码为类型。

```rust
use typenum::{U0, U1, U2, P1, P2, N1};

// U2 表示无符号2
// P2 表示有符号+2
// N1 表示有符号-1
```

∎

---

## 3. 算术运算

### 定理 3.1 (类型运算符)

> 算术在类型系统执行。

```rust
use typenum::{Sum, Prod, Unsigned};
use typenum::consts::{U2, U3};

type Five = Sum<U2, U3>;  // U5
```

∎

---

## 4. 使用场景

### 定理 4.1 (泛型数组)

> 数组大小计算。

```rust
use generic_array::GenericArray;
use typenum::consts::U16;

let arr: GenericArray<u8, U16> = GenericArray::default();
```
