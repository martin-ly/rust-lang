## 1. 引言

heapless提供:

- 固定容量集合
- 栈分配
- no_std支持
- 溢出安全

---

## 2. 固定容量

### 定理 2.1 (编译时容量)

> 容量为类型参数。

```rust
use heapless::Vec;

let mut vec: Vec<u8, 64> = Vec::new();  // 最大64元素
// 类型系统保证容量
```

∎

---

## 3. 溢出处理

### 定理 3.1 (push返回Result)

> 溢出时返回Err而非panic。

```rust
match vec.push(item) {
    Ok(()) => {},
    Err(item) => { /* 处理溢出 */ }
}
```

∎

## 4. 类型别名

### 定理 4.1 (常用容量)

> 预定义常用容量类型。

```rust
use heapless::spsc::Queue;
type U8_64 = Queue<u8, 64>;  // 64字节队列
```

∎
