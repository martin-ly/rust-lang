## 2. 启动流程

### 定理 2.1 (Reset Handler)

> 系统启动后执行的第一个函数。

```rust
#[entry]
fn main() -> ! {
    // 初始化后调用
    // 不返回
}
```

∎

## 3. 向量表

### 定理 3.1 (异常向量)

> 定义异常和中断处理函数。

```rust
#[exception]
fn HardFault(ef: &ExceptionFrame) -> ! {
    // 硬件错误处理
    loop {}
}
```

∎

---

## 4. 内存布局

### 定理 4.1 (链接脚本)

> memory.x定义内存区域。

```ld
MEMORY
{
    FLASH (rx) : ORIGIN = 0x08000000, LENGTH = 512K
    RAM (rwx) : ORIGIN = 0x20000000, LENGTH = 96K
}
```

∎

## 5. 反例

### 反例 5.1 (堆栈溢出)

```rust
// 无栈溢出保护
#[entry]
fn main() -> ! {
    let big = [0u8; 100000];  // 可能溢出!
    loop {}
}
```
