## 2. 调度器控制

### 定理 2.1 (确定性调度)

> Shuttle控制任务调度顺序。

```rust
#[test]
fn test_concurrent() {
    shuttle::check_random(
        || {
            // 此代码在Shuttle控制下运行
            spawn(async { ... });
            spawn(async { ... });
        },
        100,  // 100次随机调度
    );
}
```

∎

## 4. 可重现性

### 定理 4.1 (种子控制)

> 通过种子重现特定调度。

```rust
shuttle::check_random_with_seed(
    || { ... },
    100,
    0x12345678,  // 特定种子
);
```

∎
