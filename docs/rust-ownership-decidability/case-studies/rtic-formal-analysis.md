## 2. 任务模型

### 定理 2.1 (任务定义)

> 任务由属性宏定义。

```rust
#[app(device = stm32f4::Peripherals)]
const APP: () = {
    #[init]
    fn init() { ... }

    #[task(priority = 2)]
    fn foo() { ... }

    #[interrupt(binds = TIM2)]
    fn bar() { ... }
};
```

∎

## 3. 资源管理

### 定理 3.1 (优先级 ceiling)

> 资源锁提升优先级防止死锁。

```rust
#[resources]
struct Resources {
    #[lock_free]
    shared: u32,
}
```

∎
