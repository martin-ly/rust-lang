## 2. 延迟格式化

### 定理 2.1 (主机端格式化)

> 格式字符串在主机端处理。

```rust
defmt::info!("x = {=u8}", x);
// 只传输x值，不传输字符串
```

∎

## 3. 日志级别

### 定理 3.1 (编译时过滤)

> 日志级别在编译时确定。

```rust
// Cargo.toml
defmt = { version = "0.3", features = ["unstable-test"] }

// 编译时移除低级别日志
```

∎
