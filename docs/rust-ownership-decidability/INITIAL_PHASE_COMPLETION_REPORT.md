## 核心理论贡献

### 1. Linearizability 的严格形式化

在 Coq 中完整定义了 Payet et al. (NFM 2022) 的关键条件：

```coq
Definition Linearizable (Γ : type_env) : Prop :=
  forall x τ,
    te_lookup Γ x = Some τ ->
    forall y, In y (ty_refs τ) ->
    exists τ',
      te_lookup Γ y = Some τ' /\
      ty_rank τ > ty_rank τ'.
```

这是证明 borrow checking 终止性的**关键理论贡献**。

### 2. 完整的类型系统

实现了 Rust 核心类型系统，包括：

- 所有基础类型 (i8-i128, u8-u128, bool, char, str)
- 引用类型 (`&T`, `&mut T`) 和生命周期
- Box 类型
- 元组和结构体
- 完整的类型环境操作

### 3. 双重操作语义

定义并关联了两种操作语义：

- **大步语义** (`eval`): 适合类型保持证明
- **小步语义** (`step`): 适合并发和步数分析

### 4. 所有权安全判断

核心判断 `ownership_safe` 精确捕获了 Rust 的所有权规则。
