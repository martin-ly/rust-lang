# 15 区块链

## 模块简介

本模块系统梳理Rust在区块链领域的理论基础、核心技术与工程实践，涵盖分布式账本、共识机制、智能合约、安全性、加密算法、去中心化身份（DID）、零知识证明（ZKP）、跨链与原子交换等。通过理论分析、代码示例与工程案例，帮助读者全面掌握区块链系统的设计与实现。
