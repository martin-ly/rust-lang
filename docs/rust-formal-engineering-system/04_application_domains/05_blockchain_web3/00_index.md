# 区块链与 Web3（Blockchain & Web3）索引

## 目的

- 介绍 Rust 在区块链和 Web3 领域的应用与实践。
- 提供区块链开发、智能合约、去中心化应用的技术指导。

## 核心概念

- 区块链基础：分布式账本、共识机制、密码学
- 智能合约：以太坊、Solana、Polkadot 等平台
- 去中心化应用：DApp 开发、Web3 集成
- 加密货币：钱包、交易、挖矿
- 跨链技术：桥接、互操作性

## 与 Rust 的关联

- 性能优势：Rust 的高性能适合区块链节点开发
- 内存安全：防止智能合约中的内存相关漏洞
- 并发安全：支持高并发的区块链网络
- 生态系统：丰富的区块链开发库

## 术语（Terminology）

- 区块链（Blockchain）、智能合约（Smart Contract）
- 共识机制（Consensus Mechanism）、挖矿（Mining）
- 去中心化（Decentralization）、Web3
- 跨链（Cross-chain）、桥接（Bridge）
