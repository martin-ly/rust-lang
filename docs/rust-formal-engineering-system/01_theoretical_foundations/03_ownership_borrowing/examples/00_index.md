## 目的

本目录包含所有权与借用相关的代码示例，用于演示所有权系统和借用检查器。

## 示例分类

- **所有权转移示例**: 所有权在不同作用域间的转移
- **借用规则示例**: 借用检查器的规则演示
- **可变借用示例**: 可变借用的使用和限制
- **生命周期示例**: 生命周期注解的使用
- **作用域示例**: 变量作用域和所有权
- **借用检查器示例**: 借用检查器的行为演示
