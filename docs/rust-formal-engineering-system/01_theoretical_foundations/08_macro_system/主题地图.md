## 3. 术语体系

### 基础术语

- **声明宏（Declarative Macro）**：使用 `macro_rules!` 定义的宏
- **过程宏（Procedural Macro）**：使用函数式、派生或属性宏定义的宏
- **宏展开（Macro Expansion）**：将宏调用转换为实际代码的过程
- **卫生宏（Hygienic Macro）**：避免变量捕获的宏系统

### 进阶术语

- **TokenStream**：宏处理的令牌流
- **AST（Abstract Syntax Tree）**：抽象语法树
- **Hygiene**：宏的卫生性机制
- **Gensym**：生成唯一标识符
