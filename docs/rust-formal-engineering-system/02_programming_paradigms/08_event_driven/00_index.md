## 术语

- **事件驱动（Event-Driven）**: 程序执行由事件触发的编程范式。
- **事件（Event）**: 程序中发生的重要事情。
- **事件循环（Event Loop）**: 等待和分发事件的循环。
- **事件处理器（Event Handler）**: 处理特定事件的函数。

---

## 核心概念

- **事件系统**: 事件的定义、分发和处理。
- **事件循环**: 异步事件循环实现。
- **事件总线**: 事件发布和订阅机制。
- **事件过滤**: 事件的选择性处理。

## 设计建议

- 设计清晰的事件类型。
- 使用事件总线解耦组件。
- 合理处理事件优先级。
- 实现事件过滤机制。

---

## 常见陷阱

- 事件处理顺序不当。
- 事件循环阻塞。
- 事件丢失或重复。
