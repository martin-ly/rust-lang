## 研究方向

### 形式化验证

- **定理证明**: Coq、Isabelle、Lean
- **模型检测**: Kani、MIRI
- **符号执行**: 路径覆盖、约束求解

### 类型系统研究

- **高级类型**: 依赖类型、线性类型
- **类型推断**: Hindley-Milner、局部类型推断
- **型变**: 协变、逆变、不变

### 并发与并行

- **内存模型**: 顺序一致性、释放-获取语义
- **无锁编程**: CAS、内存排序
- **分布式系统**: 一致性协议、容错
