## 🚀 使用指南

### 本地开发

```bash
# 启动开发服务器
cd book
mdbook serve --open

# 访问 http://localhost:3000
```

### 构建发布

```bash
# 构建静态文件
cd book
mdbook build

# 输出目录: book/book/
```

### 部署到 GitHub Pages

**一次性设置**:

1. 进入 GitHub 仓库 Settings → Pages
2. Source 选择 "GitHub Actions"
3. 保存设置

**后续使用**:

```bash
# 推送即可自动部署
git add .
git commit -m "Update documentation"
git push
```
