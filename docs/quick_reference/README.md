## 🚀 快速开始

### 推荐阅读顺序

1. **初学者**: 所有权系统 → 类型系统 → 错误处理 → 泛型编程
2. **中级开发者**: 类型系统 → 错误处理 → 异步编程 → 线程与并发
3. **高级开发者**: 异步编程 → 线程与并发 → 宏系统 → 泛型编程
4. **特定需求**: 直接查找对应速查卡

### 使用建议

- 📖 **打印版本**: 可以打印速查卡作为桌面参考
- 💻 **代码编辑器**: 在编辑器中打开，随时查阅
- 🔖 **书签**: 添加到浏览器书签，快速访问
