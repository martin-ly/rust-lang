## 主题概述

Rust宏系统提供了强大的代码生成和元编程能力，包括声明宏、过程宏和属性宏。本主题涵盖：

- **宏基础**：声明宏、过程宏、属性宏
- **宏语法**：宏规则、模式匹配、语法扩展
- **宏编译**：宏展开、编译时执行、代码生成
- **宏应用**：代码生成、DSL实现、编译时计算

本文档使用以下数学符号：
