# 分布式系统理论

## 1. CAP定理与BASE模型

- CAP定理：一致性(Consistency)、可用性(Availability)、分区容错性(Partition Tolerance)不可兼得
- BASE模型：基本可用、软状态、最终一致性

## 2. 一致性理论

- 强一致性、最终一致性、因果一致性
- 分布式事务与两阶段/三阶段提交

## 3. 分区容错与共识算法

- 网络分区下的系统可用性
- Raft、Paxos、PBFT等共识算法

## 4. 工程案例

