## 核心代码示例

```rust
// 泛型结构体体体体定义
struct Point<T> {
    x: T,
    y: T,
}

fn main() {
    let p1 = Point { x: 1, y: 2 };         // T为i32
    let p2 = Point { x: 1.0, y: 2.0 };     // T为f64
    println!("p1: ({}, {}), p2: ({}, {})", p1.x, p1.y, p2.x, p2.y);
}
```

- `struct Point<T>` 通过类型参数T实现字段类型抽象。
- 可为不同类型实例化，提升代码复用性。
