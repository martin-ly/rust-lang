## 模块概述

过程式编程语义模块是Rust范式语义的重要组成部分，涵盖了过程式编程的完整语义定义，包括过程、状态管理、副作用和命令式控制流等核心概念。本模块建立了严格的理论基础，为Rust语言的过程式编程提供了形式化的语义定义。
