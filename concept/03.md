## 关键交叉概念

```text
L3 内部交叉
    │
    ├──→ 异步 = 单线程并发（Concurrency ↔ Async 等价/特化关系）
    ├──→ Unsafe 突破所有 safe 保证（Unsafe ↔ Concurrency/Async/Macros）
    └──→ 宏生成辅助并发与异步代码（Macros → Concurrency/Async）
```

## L3 → L4 形式化映射

| L3 概念 | L4 形式化对应 | 映射精度 |
|:---|:---|:---|
| 并发安全 | 并发分离逻辑（CSL）+ Iris Protocols | 近似 |
| `Future`/`async` | 效果系统 / 续体（Continuation）| 部分形式化 |
| `Pin` 不动性 | 线性时序逻辑（LTL）| 待形式化 |
| 宏卫生性 | 元类型论 / 准引用（Quasiquote）| 近似 |
