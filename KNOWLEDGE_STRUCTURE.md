## 📚 模块知识图谱

### C01 所有权系统

```
所有权系统
│
├─ 核心概念
│   ├─ 所有权规则
│   │   ├─ 每个值有所有者
│   │   ├─ 只有一个所有者
│   │   └─ 所有者离开作用域，值被释放
│   │
│   ├─ 移动语义
│   │   ├─ Copy trait
│   │   ├─ Drop trait
│   │   └─ 所有权转移
│   │
│   └─ 借用机制
│       ├─ 不可变借用 &T
│       ├─ 可变借用 &mut T
│       └─ 借用规则
│
├─ 生命周期
│   ├─ 显式生命周期
│   ├─ 生命周期省略规则
│   ├─ 生命周期标注
│   └─ 生命周期边界
│
└─ 智能指针
    ├─ Box<T>
    ├─ Rc<T>
    ├─ RefCell<T>
    ├─ Arc<T>
    ├─ Mutex<T>
    └─ LazyLock<T>
```
