# 00 - 开始学习 Rust

> **从这里开始你的 Rust 之旅**

## 🎯 本模块学习目标

完成本模块后，你将能够：

- [ ] 安装 Rust 工具链
- [ ] 编写并运行第一个 Rust 程序
- [ ] 理解 Rust 的核心设计理念
- [ ] 选择合适的学习路径

## 📚 学习路径

按顺序学习以下文档：

1. [installation.md](installation.md) - 安装和环境配置
2. [hello_world.md](hello_world.md) - 第一个程序
3. [rust_philosophy.md](rust_philosophy.md) - Rust 设计哲学
4. [learning_roadmap.md](learning_roadmap.md) - 学习路线图

## ⏱️ 预计时间

- 安装配置: 15-30 分钟
- Hello World: 15 分钟
- 理解哲学: 30 分钟
- **总计**: 约 1-1.5 小时

## 🎓 预备知识

- 基本的命令行操作
- 至少一门其他编程语言经验（推荐）
- 对系统编程有兴趣

## ✅ 完成检查清单

- [ ] 成功运行 `rustc --version`
- [ ] 成功运行 `cargo --version`
- [ ] 成功编译并运行 Hello World
- [ ] 理解所有权的基本概念
- [ ] 确定个人学习路径

## 🚀 下一步

完成本模块后，进入 [01_fundamentals/](../01_fundamentals/) 学习 Rust 基础。
