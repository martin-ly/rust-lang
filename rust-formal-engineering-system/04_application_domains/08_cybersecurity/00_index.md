# 网络安全（Cybersecurity）索引

## 目的

- 介绍 Rust 在网络安全领域的应用与实践。
- 提供安全编程、密码学、漏洞防护的技术指导。

## 核心概念

- 密码学：对称加密、非对称加密、哈希函数
- 网络安全：TLS/SSL、VPN、防火墙
- 漏洞防护：缓冲区溢出、内存泄漏、竞态条件
- 安全编程：输入验证、错误处理、权限控制
- 威胁建模：风险评估、安全设计

## 与 Rust 的关联

- 内存安全：防止缓冲区溢出、悬垂指针等漏洞
- 并发安全：防止数据竞争、竞态条件
- 性能优势：高效的加密算法实现
- 类型安全：编译时保证类型正确性

## 术语（Terminology）

- 密码学（Cryptography）、加密（Encryption）
- 漏洞（Vulnerability）、攻击（Attack）
- 安全编程（Secure Programming）、威胁建模（Threat Modeling）
- 渗透测试（Penetration Testing）、安全审计（Security Audit）
