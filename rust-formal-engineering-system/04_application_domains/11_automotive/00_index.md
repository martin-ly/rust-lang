# 汽车（Automotive）索引

## 目的

- 介绍 Rust 在汽车领域的应用与实践。
- 提供自动驾驶、车载系统、汽车网络的技术指导。

## 核心概念

- 自动驾驶：感知、决策、控制
- 车载系统：信息娱乐、导航系统
- 汽车网络：CAN、LIN、以太网
- 传感器融合：多传感器数据融合
- 实时系统：硬实时、软实时
- 功能安全：ISO 26262 标准
- 网络安全：车载网络安全
- 边缘计算：车载边缘计算

## 与 Rust 的关联

- 内存安全：防止汽车系统崩溃
- 性能优势：实时控制系统
- 并发安全：多任务车载系统
- 跨平台：支持多种汽车硬件

## 术语（Terminology）

- 汽车（Automotive）、自动驾驶（Autonomous Driving）
- 车载系统（In-Vehicle Systems）、汽车网络（Automotive Networks）
- 传感器融合（Sensor Fusion）、实时系统（Real-Time Systems）
- 功能安全（Functional Safety）、网络安全（Cybersecurity）
