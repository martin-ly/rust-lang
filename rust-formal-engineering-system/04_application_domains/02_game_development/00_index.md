# 游戏开发（Game Development）索引

## 目的

- 介绍 Rust 在游戏开发领域的应用与实践。
- 提供游戏引擎、图形渲染、物理模拟的技术指导。

## 核心概念

- 游戏引擎：渲染引擎、物理引擎、音频引擎
- 图形渲染：OpenGL、Vulkan、DirectX 集成
- 物理模拟：碰撞检测、刚体动力学
- 音频处理：3D 音频、音效处理
- 输入处理：键盘、鼠标、手柄输入
- 网络多人：实时多人游戏
- 资源管理：纹理、模型、音频资源
- 场景管理：场景图、空间分割

## 与 Rust 的关联

- 性能优势：低延迟、高帧率游戏
- 内存安全：防止游戏崩溃
- 并发安全：多线程游戏逻辑
- 跨平台：支持多平台部署

## 术语（Terminology）

- 游戏引擎（Game Engine）、渲染引擎（Rendering Engine）
- 物理引擎（Physics Engine）、音频引擎（Audio Engine）
- 碰撞检测（Collision Detection）、刚体动力学（Rigid Body Dynamics）
- 场景图（Scene Graph）、空间分割（Spatial Partitioning）
