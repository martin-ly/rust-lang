# 宏系统（Macro System）索引

- 宏种类：声明式宏（macro_rules!）、过程宏（派生、属性、函数样式）

- 设计建议：语义清晰、错误信息友好、最小可见性、避免过度魔法

- `proc-macro` crate 结构与测试策略
- `syn`/`quote` 解析与生成
- 错误定位与 `Span` 使用
