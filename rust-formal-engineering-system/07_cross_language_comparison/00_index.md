# 跨语言比较（Cross-Language Comparison）索引

## 目的

- 从类型安全、并发模型、内存管理、生态成熟度等维度对比 Rust 与其他语言。

## 对象与维度

- 维度：类型系统/错误处理/并发与异步/工具链/性能/部署/学习曲线

## 建议阅读顺序

1) 并发与内存安全对比
2) 类型系统与抽象能力
3) 工具链与生态成熟度
4) 典型场景的实现复杂度与运维成本
