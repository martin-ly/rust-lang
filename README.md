## 🚀 快速开始

### 环境准备

```bash
# 1. 安装 Rust (推荐使用 rustup)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 2. 验证安装
rustc --version
cargo --version

# 3. 克隆项目
git clone <repository-url>
cd rust-lang

# 4. 构建项目
```

### 贡献流程

```bash

# 2. 创建分支

# 3. 进行修改
# ... 编辑文件 ...

# 4. 提交改动
git add .
git commit -m "描述你的改动"

# 5. 推送分支
git push origin feature/your-feature

```
