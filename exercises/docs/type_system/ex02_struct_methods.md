# 结构体与方法

**主题**: Type System
**难度**: Easy
**练习编号**: ex02_struct_methods

---

## 题目描述

实现 Rectangle 结构体，包含面积计算和缩放方法。

## 代码位置

- 代码框架: `exercises/src/type_system/ex02_struct_methods.rs`
- 测试用例: 同一文件内的 `#[cfg(test)]` 模块

## 学习目标

- 理解 type/system 的核心概念
- 能够独立编写通过测试的实现
- 掌握相关的 Rust 语法和惯用法

## 运行测试

```bash
cd exercises
cargo test type_system::ex02_struct_methods
```

或运行整个主题的测试：

```bash
cd exercises
cargo test type_system::
```
