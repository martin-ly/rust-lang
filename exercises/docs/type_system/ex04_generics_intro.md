# 泛型入门

**主题**: Type System
**难度**: Medium
**练习编号**: ex04_generics_intro

---

## 题目描述

实现泛型二分查找和切片最大值函数。

## 代码位置

- 代码框架: `exercises/src/type_system/ex04_generics_intro.rs`
- 测试用例: 同一文件内的 `#[cfg(test)]` 模块

## 学习目标

- 理解 type/system 的核心概念
- 能够独立编写通过测试的实现
- 掌握相关的 Rust 语法和惯用法

## 运行测试

```bash
cd exercises

```
