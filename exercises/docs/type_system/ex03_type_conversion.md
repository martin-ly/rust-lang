# 类型转换

**主题**: Type System
**难度**: Easy
**练习编号**: ex03_type_conversion

---

## 题目描述

为自定义温度类型实现 From/Into 特质。

## 代码位置

- 代码框架: `exercises/src/type_system/ex03_type_conversion.rs`
- 测试用例: 同一文件内的 `#[cfg(test)]` 模块

## 学习目标

- 理解 type/system 的核心概念
- 能够独立编写通过测试的实现
- 掌握相关的 Rust 语法和惯用法

## 运行测试

```bash
cd exercises
cargo test type_system::ex03_type_conversion
```

或运行整个主题的测试：

```bash
cd exercises
cargo test type_system::
```
