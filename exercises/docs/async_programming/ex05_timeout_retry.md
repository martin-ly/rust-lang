# 超时与重试

**主题**: Async Programming
**难度**: Hard
**练习编号**: ex05_timeout_retry

---

## 题目描述

实现带有超时和重试机制的异步操作。

## 代码位置

- 代码框架: `exercises/src/async_programming/ex05_timeout_retry.rs`
- 测试用例: 同一文件内的 `#[cfg(test)]` 模块

## 学习目标

- 理解 async/programming 的核心概念
- 能够独立编写通过测试的实现
- 掌握相关的 Rust 语法和惯用法

## 运行测试

```bash
cd exercises
cargo test async_programming::ex05_timeout_retry
```

或运行整个主题的测试：

```bash
cd exercises
cargo test async_programming::
```
