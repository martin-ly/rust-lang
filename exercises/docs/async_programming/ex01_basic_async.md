# 基础 async/await

**主题**: Async Programming
**难度**: Easy
**练习编号**: ex01_basic_async

---

## 题目描述

实现简单的异步函数，理解 .await 用法。

## 代码位置

- 代码框架: `exercises/src/async_programming/ex01_basic_async.rs`
- 测试用例: 同一文件内的 `#[cfg(test)]` 模块

## 学习目标

- 理解 async/programming 的核心概念
- 能够独立编写通过测试的实现
- 掌握相关的 Rust 语法和惯用法

## 运行测试

```bash
cd exercises
cargo test async_programming::ex01_basic_async
```
