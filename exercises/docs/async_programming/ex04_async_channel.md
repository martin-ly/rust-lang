# 异步通道

**主题**: Async Programming  
**难度**: Medium  
**练习编号**: ex04_async_channel

---

## 题目描述

使用 tokio::sync::mpsc 进行异步消息传递。

## 代码位置

- 代码框架: `exercises/src/async_programming/ex04_async_channel.rs`
- 测试用例: 同一文件内的 `#[cfg(test)]` 模块

## 学习目标

- 理解 async/programming 的核心概念
- 能够独立编写通过测试的实现
- 掌握相关的 Rust 语法和惯用法
