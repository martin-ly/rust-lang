# Result 与 Option

**主题**: Error Handling  
**难度**: Easy  
**练习编号**: ex01_result_option

---

## 题目描述

掌握 Result 和 Option 的基本操作与转换。

## 代码位置

- 代码框架: `exercises/src/error_handling/ex01_result_option.rs`
- 测试用例: 同一文件内的 `#[cfg(test)]` 模块

## 学习目标

- 理解 error/handling 的核心概念
- 能够独立编写通过测试的实现
- 掌握相关的 Rust 语法和惯用法

## 运行测试

```bash
cd exercises
cargo test error_handling::ex01_result_option
```
