# 字符串 Slice

**主题**: Ownership Borrowing  
**难度**: Easy  
**练习编号**: ex02_string_slice

---

## 题目描述

掌握字符串 slice 的使用，实现 first_word 函数。

## 代码位置

- 代码框架: `exercises/src/ownership_borrowing/ex02_string_slice.rs`
- 测试用例: 同一文件内的 `#[cfg(test)]` 模块

## 学习目标

- 理解 ownership/borrowing 的核心概念
- 能够独立编写通过测试的实现
- 掌握相关的 Rust 语法和惯用法

## 运行测试

```bash
cd exercises
cargo test ownership_borrowing::ex02_string_slice
```

或运行整个主题的测试：

```bash
cd exercises
cargo test ownership_borrowing::
```
