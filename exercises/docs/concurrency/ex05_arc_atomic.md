# Arc + Atomic

**主题**: Concurrency
**难度**: Medium
**练习编号**: ex05_arc_atomic

---

## 题目描述

使用 AtomicUsize 实现无锁计数器。

## 代码位置

- 代码框架: `exercises/src/concurrency/ex05_arc_atomic.rs`
- 测试用例: 同一文件内的 `#[cfg(test)]` 模块

## 学习目标

- 理解 concurrency 的核心概念
- 能够独立编写通过测试的实现
- 掌握相关的 Rust 语法和惯用法
