# MPSC 通道

**难度**: Easy

## 题目描述

使用 mpsc 通道在线程间传递消息。

## 代码位置

- 代码框架: `exercises/src/concurrency/ex03_channel_mpsc.rs`

- 掌握相关的 Rust 语法和惯用法
