# 特质组合

**难度**: Hard

## 题目描述

使用 supertrait 组合 Display 和 PartialOrd。

## 代码位置

- 代码框架: `exercises/src/generics_traits/ex05_trait_composition.rs`
- 测试用例: 同一文件内的 `#[cfg(test)]` 模块

## 学习目标

- 理解 generics/traits 的核心概念
- 能够独立编写通过测试的实现
- 掌握相关的 Rust 语法和惯用法
