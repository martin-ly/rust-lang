# 运算符重载

**主题**: Generics Traits
**难度**: Medium
**练习编号**: ex03_operator_overload

---

## 题目描述

为二维向量实现 Add 和 Sub 运算符。

## 代码位置

- 代码框架: `exercises/src/generics_traits/ex03_operator_overload.rs`
- 测试用例: 同一文件内的 `#[cfg(test)]` 模块

## 学习目标

- 理解 generics/traits 的核心概念
- 能够独立编写通过测试的实现
- 掌握相关的 Rust 语法和惯用法
