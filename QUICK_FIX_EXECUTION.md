## 🚀 快速执行步骤

### 步骤1: 编译工具

```bash
# 进入工具目录
cd tools

# 编译工具
cargo build --release

# 返回根目录
cd ..
```

### 步骤2: 执行试运行

```bash
# 给脚本执行权限
chmod +x scripts/quick_batch_fix.sh

# 执行试运行
./scripts/quick_batch_fix.sh ./formal_rust --dry-run
```

### 步骤3: 执行正式修复

```bash
# 执行正式修复（包含备份）
./scripts/quick_batch_fix.sh ./formal_rust

# 或者不创建备份（谨慎使用）
./scripts/quick_batch_fix.sh ./formal_rust --no-backup
```
