## ❓ 还有拼写错误？

### 临时禁用（单个文件）

在文件顶部添加：

```markdown
<!-- cSpell:disable -->
```

### 添加新词汇

编辑 `.vscode/settings.json`，在相应类别下添加：

```json
{
  "cSpell.words": [
    // ... 现有词汇 ...
    "你的新术语"
  ]
}
```
