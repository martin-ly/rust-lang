## 🔧 自定义配置

### 添加新术语

编辑 `.vscode/settings.json`：

```json
{
  "cSpell.words": [
    // ... 现有 300+ 术语 ...

    // 添加你的新术语
    "YourNewTerm1",
    "YourNewTerm2"
  ]
}
```

### 临时禁用检查

在文件顶部添加：

```markdown
<!-- cSpell:disable -->

你的文档内容...

<!-- cSpell:enable -->
```
