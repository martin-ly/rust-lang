# Representations 目录

> **定位**: Rust 知识体系的**表征形式**汇总区：同一概念在不同抽象层次、不同学习阶段、不同应用场景下的多种表示方式。
> **Rust 版本**: 1.96.0+ (Edition 2024)
> **更新**: 随概念体系持续演进

## 什么是知识表征

知识表征（Knowledge Representation）回答的是：**同一个 Rust 概念，如何以不同形式呈现给不同受众**？

| 表征维度 | 示例 |
|---|---|
| **抽象层次** | 直观类比 → 操作定义 → 形式化语义 |
| **学习阶段** | 入门示例 → 工程模式 → 专家级推理树 |
| **媒介形式** | 文本、代码、图表、定理、决策树、故障树 |
| **应用场景** | 面试题、代码审查、形式化验证、教学讲义 |

---

## 与主轨的关系

- `concept/` 提供概念的** canonical 定义**。
- `knowledge/` 提供按难度组织的**学习路径**。
- `content/representations/` 提供概念的**多模态、多粒度表示**，用于教学设计、评估和知识图谱构建。
